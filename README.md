- PC：http://kefu.zhaoey.club/
- 移动端：http://kefu.zhaoey.club/mobile

## 3.技术架构
- SpringBoot v2.1.6
- Netty
- Websocket
- Layim
- MybatisPlus
- Mysql
- SpringSecurity
- Thymeleaf
- Druid

## 4.支持的功能
- 单聊
- 群聊
- 聊天记录
- 状态切换
- 签名
- 消息提醒
- 未读消息数量显示
- 搜索
- 加好友
- 加群
- 申请审核
- 断线重连
- 服务端空闲检测
- 客户端心跳检测
- 在线状态实时更新(头像点亮与置灰)
- 离线消息推送设计
